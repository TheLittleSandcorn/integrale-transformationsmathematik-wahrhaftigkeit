## Kurzbeschreibung

Dieses Repository dokumentiert einen konsolidierten Verlauf zur **Integralen Transformationsmathematik**: von Vorantike, Antike, Mittelalter, Renaissance, Dampfmaschine, Elektrizität, Weltkriegen, Computer, Microchip und Petrodollar bis zu Artificial Intelligence, Transformer-Modellen, AI-Governance und Wahrhaftigkeitsanalyse.

Der Kern ist eine strukturierte Dokumentation aus:

- `00_MASTER_INDEX` mit allen 27 Arbeitsschritten
- `01_Wahrhaftigkeit` mit Methodik, Anti-Mythos-Regeln und Statuscodes
- `02_Zeitalter_Index`
- `03_Formel_Index`
- 27 einzelnen TOP-Tabellen: `01_TOP_01` bis `27_TOP_27`
- `31_Prüfwerte` mit Manifest-Daten
- SHA-256-Prüfsummen in Manifest-Dateien

## Wahrhaftigkeitsstatus

Dieses Repository ist **kein peer-reviewter historischer Endbeweis**. Es ist ein **transparentes, versioniertes, AI-assistiertes Forschungs- und Dokumentationsartefakt**.

Wahrheitsgrenze:

> Frühere Chatteile waren teilweise als „Skipped“ ausgeblendet. Diese Inhalte werden nicht als wortwörtliches Volltranskript behauptet, sondern als konsolidierte, methodisch markierte Dokumentation.

## Statuscodes

| Code | Bedeutung |
|---|---|
| H | Historisch belegt / institutionell oder quellenbasiert dokumentiert |
| F | Mathematisch korrekt als Formel / Definition / Struktur |
| R | Moderne Rekonstruktion oder analytisches Modell auf historische Praxis angewandt |
| K | Kritische, ethische, kulturelle oder gesellschaftliche Einordnung |
