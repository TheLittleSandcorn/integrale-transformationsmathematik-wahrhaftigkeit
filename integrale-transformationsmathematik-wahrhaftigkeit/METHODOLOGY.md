# METHODOLOGY.md — Methodik

## Ziel

Das Projekt dokumentiert den Verlauf einer mehrstufigen Transformation:

Zeitalter → Weltkultur → TOP-Inhalt → Formel → Status → Vernetzung → Kritik/Ethik

## 7-Spalten-Standard

| Spalte | Funktion |
|---|---|
| Epoche | chronologische Einordnung |
| Weltkultur / Raum | kultureller und geografischer Kontext |
| Transformation | wissenschaftlicher, technischer oder sozialer Sprung |
| Formel / Struktur | mathematische Struktur oder moderne Rekonstruktion |
| Quellenstatus | H/F/R/K |
| Wirkung | technische, soziale, kulturelle oder politische Wirkung |
| Kritik / Ethik | Grenzen, Risiken, Machtfolgen |

## Statuscodes

- H = historisch belegt
- F = mathematisch korrekt
- R = moderne Rekonstruktion
- K = kritische / ethische Einordnung

## Quellenprinzip

1. Primärquellen und offizielle Institutionen bevorzugen.
2. Bei historischen Aussagen Datierung und Kontext nennen.
3. Bei moderner Rekonstruktion keine historische Notation vortäuschen.
4. Bei AI-Aussagen zwischen Modellfähigkeit, Benchmark und Wahrheit unterscheiden.
5. Bei gesellschaftlichen Anwendungen ethische Grenzen dokumentieren.

## Kernformel

```text
Wahrhaftigkeit = Fakt + Formel + Kontext + Quelle + Ethik - Mythos - Scheingenauigkeit
```
